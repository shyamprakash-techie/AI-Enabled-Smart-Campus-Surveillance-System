# AI-Enabled-Smart-Campus-Surveillance-System

## Description
Edge-based CCTV analytics system for detecting presence, crowd formation,
and unusual activities in campus environments.

## Features
- Local Edge AI video processing
- Real-time alert generation
- Security monitoring dashboard
- Privacy-aware design
- No cloud dependency

## How to Run
1. pip install -r requirements.txt
2. python api_server.py
3. python edge_ai.py
4. Open http://localhost:5000
